# Image OCR Web API

Go to [Web API](https://ocr-web-api.heroku.ctdn.dev)

-   Flask
-   Pytesseract (`pip install pytesseract`)
-   Pillow (`pip install Pillow`)
-   Tesseract-ocr (`sudo apt install tesseract-ocr`)

### Features

-   [x] Image to Text
-   [x] Support Languages, English & Khmer (`eng` and `khm`)
